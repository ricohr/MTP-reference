# WhiteBalance

## Device Prop Code

0x5005

## Overview

Set or fetch white balance.

## Support value

| Value | Description |
|:---|:---|
| 1 | Manual ([RGBGain](../property/0x5006_RGBGain.md) is enabled) |
| 2 | AUTO |
| 32775 | ColorTemperature ([ColorTemperature](../property/0xD81C_ColorTemperature.md) is enabled) |
