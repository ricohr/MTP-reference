# RGBGain

## Device Prop Code

0x5006

## Overview

Set or fetch RGB values.<BR>
([WhiteBalance](../property/0x5005_WhiteBalance.md) is only enabled in Manual

## Support value

Set RGB values in "R:G:B" string format as numbers.
G is fixed at 100.

| Value | Description |
|:---|:---|
| R | 0～500(1 unit) |
| G | 100(Fixed) |
| B | 0～500(1 unit) |
